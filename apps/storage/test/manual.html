3. 选项 Playground（namespace / sliding / raw / memoized / codec）

namespace codec 口令
key value ttl(ms)

提示：勾选/改动上方任一选项即自动应用（无需手动点「应用选项」）；namespace、codec 改动后立刻对随后的 set/get 生效。
① 开 codeable + 填口令后 set，再点「查看底层原始值」可看到被混淆、非明文（且带 namespace 前缀）。
② sliding 必须配合 ttl：sliding 是「读命中时按 ttl 重算 expireAt」，不填 ttl 就没有 expireAt，自然不会更新。 填 ttl → set → 隔一会 → get → 再「查看底层原始值」，可见 expireAt 前移。
③ 注意：改了 codec 口令后，之前用旧口令写入的数据将解不开（读出为 null）——这是预期行为。
