picman 三段渐进加载 demo

运行前先放一张较大的动图到 examples/big.gif(建议 > 500KB,便于在限速下观察三段时间轴), 再执行 pnpm build && node --experimental-strip-types examples/serve.ts --throttle 51200, 浏览器打开 http://localhost:8787。
